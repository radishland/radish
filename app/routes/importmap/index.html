The importmap

rounded value:

For the sake of the example, here we use a helper library to round a number to 1 decimal place

In dev mode the importmap resolves modules from the node_modules folder, in production, it resolves modules from CDNs

To generate the importmap in dev mode run: deno task generate --importmap --dev

To generate the importmap in prod mode run: deno task generate --importmap --prod
